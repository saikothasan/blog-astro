# Getting Started with Astro

Astro is a modern web framework that offers an incredible developer experience while delivering lightning-fast performance. In this guide, we'll explore the key features that make Astro stand out.

## Why Choose Astro?

- **Performance First**: Ships zero JavaScript by default
- **Component Islands**: Use your favorite UI frameworks
- **Content Collections**: Type-safe content management
- **Server-first API design**: Optimal loading strategies

## Quick Start

```bash
# Create a new project
npm create astro@latest

# Install dependencies
npm install

# Start development server
npm run dev
```

## Key Features

1. **Component-Based Architecture**
2. **Markdown/MDX Support**
3. **Built-in Image Optimization**
